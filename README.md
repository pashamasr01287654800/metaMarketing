# metaMarketing

MetaMarketing is a lightweight PHP-based social media preview generator that creates custom Open Graph and Twitter Card pages with instant redirects.

Generate shareable links with custom titles, descriptions, and preview images for platforms like Facebook, Twitter/X, WhatsApp, Discord, Telegram, and more.

---

Features

- Custom Open Graph meta tags
- Twitter Card support
- Instant auto redirect
- Lightweight single-file PHP setup
- No database required
- Generates static HTML preview pages
- Simple and fast deployment
- Compatible with most hosting providers

---

Use Cases

- Social media marketing
- Link preview customization
- Campaign branding
- Content sharing optimization
- SEO and engagement testing

---

How It Works

MetaMarketing generates lightweight HTML pages containing:

- Open Graph metadata
- Twitter Card metadata
- Instant redirect to the target URL

When a link is shared, social media crawlers read the metadata and display the custom preview image, title, and description while visitors are automatically redirected to the original destination.

---

Run on Any PHP Server

MetaMarketing is designed to work on almost any standard PHP hosting environment.

You can deploy it on:

- Shared Hosting
- VPS Servers
- Apache Servers
- Nginx + PHP-FPM
- XAMPP
- Localhost
- Cloud Hosting Providers

Quick Setup

1. Upload the PHP file to your web server
2. Make sure PHP is enabled
3. Open the script in your browser
4. Start generating custom preview links instantly

Example:

https://yourdomain.com/index.php

No database setup, frameworks, or complex configuration required.
